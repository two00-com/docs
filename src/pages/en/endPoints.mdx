# End Points - Uptime Monitoring

## What is an Uptime Monitor used for?

An **uptime monitor** checks whether a web page, a web server or an endpoint is accessible on it or not. This means that if there is a problem on your server, you will be notified that your website or web server is no longer accessible or no longer responds with the usual status code.
