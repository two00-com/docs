# Endpunkte - Uptime Monitoring

## Wozu wird ein Uptime Monitor verwendet?

Ein **Uptime Monitor** überprüft ob eine Webseite, ein Webserver, bzw. ein Endpunkt darauf zugänglich ist oder nicht. Das heißt sollte ein Problem auf Ihrem Server bestehen werden Sie benachrichtigt das Ihre Webseite, bzw. Ihr Webserver nicht mehr erreichbar ist bzw. nicht mehr mit dem sonst üblichen Status Code antwortet.
