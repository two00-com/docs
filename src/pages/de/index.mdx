# Einführung

## Was ist two00.com?

two00.com ist eine Applikation die Software Entwicklern bzw. Unternehmen dabei unterstützt alle ihre Applikationen, Systeme und Servertasks zu Überwachen und Sie zu benachrichtigen sollte etwas nicht richtig funktionieren, damit Sie mehr Zeit haben sich um wichtigere Dinge - wie z.B. dem Entwickeln ihrer Software Applikationen, Systeme oder Servertasks - zu kümmern.

### Warum ist das wichtig?

Ich gebe Ihnen ein Beispiel:

Sie haben ein Unternehmen das einen Webshop betreibt und möchten natürlich das Ihre Kunden immer die Möglichkeit haben die Waren die Sie im Warenkorb abgelegt haben auch real bestellen zu können.

Ihr Webshop hat folgende Metriken:

- 100 Besucher pro Stunde
- 10% Warenkorb Conversion = 10 Besucher legen Produkte in Ihren Warenkorb
- 40% Bestell Conversion = 4 Besucher kaufen die Produkte die Sie in Ihren Warenkorb abgelegt haben
- Ein Nutzer gibt durchschnittlich 100€ pro Einkauf aus = 400€ pro Stunde

Stellen Sie sich nun vor ein Teil Ihrer Applikation fällt aus und Die Besucher können zwar Produkte in den Warenkorb legen aber den Bestellvorgang nicht mehr abschließen.

Ihnen fällt der Fehler erst am nächsten Tag auf als einer der Besucher ein Email an den Kundenservice geschrieben hat. Sie können den Fehler innerhalb von 2 Stunden beheben, aber in der Zwischenzeit sind schon 13 Stunden vergangen und damit haben Sie einen Verlust von € 5.300,00 in kauf nehmen müssen.

Dies wäre nicht passiert wenn Sie einen Uptime Monitor für Ihre Applikation eingerichtet hätten. Wir hätten Sie sofort informiert wenn Ihre Applikation nicht mehr richtig funktioniert. Demnach hätten Sie das problem vermutlich auch in kürzester Zeit lösen können und mindestens 4.500€ mehr Umsatz gemacht.

## Welche Monitore kann ich mit two00.com aufsetzen?

two00.com erlaubt Ihnen zurzeit 3 verschiedene Arten von Monitore aufzusetzen:

1. Server Monitore
2. Uptime Monitore
3. Cronjob Monitore

Ein Server Monitor wird auf Ihrem Server eingerichtet und gibt two00.com einmal in der Minute einen Bericht über den aktuellen Gesundheitszustandes ihres Servers. Sollte es Probleme geben benachrichtigen wir sie sofort.

Ein Uptime Monitor wird auf unseren Servern eingerichtet und erlaubt es Endpunkte (URLs) einmal in der Minute zu pingen. Sollte dieser Endpunkt nicht den erwarteten Status Code berichten, benachrichtigen wir Sie unverzüglich.

Ein Cronjob Monitor wird auf Ihrem Server eingerichtet. Jeder Task der auf Ihrem Server in einem geregelten Intervall läuft (im Regelfall ein Cronjob) kann damit überwacht werden. Sollte dieser Tasks nicht in der gewünschten Zeit fertig sein, benachrichtigen wir Sie unverzüglich.
